#Libflame checkcpp execution script

##Requirements
* Install latest version of python from python.org(preferably python 3.5 or greater)
* Add python path and scripts path to the environment variable path

#Copy all the files present in <src>/windows/tests directory to the directory where testcpp executables are present
#Open the command prompt and execute the python script
For example:
python checkcpp_test.py

#Output can be seen on the command prompt and in the testcpp_output.txt log file
